Soal — Hastad's Broadcast Attack

Temukan plaintext M yang disembunyikan!

Sebuah pesan rahasia M dikirim ke 3 penerima berbeda menggunakan RSA dengan exponent publik e = 3. Berikut data yang berhasil disadap:

PenerimaModulus (N)Ciphertext (C)
Alice3326
Bob8540
Carol9134

e = 3  |  Cᵢ = M³ mod Nᵢ  |  Cari M!

1 / 7
Langkah 1 — Verifikasi syarat serangan

Cek apakah semua syarat terpenuhi

Hastad Attack hanya bisa dilakukan jika 3 syarat ini terpenuhi:

Syarat 1: Exponent publik kecil → e = 3 ✓
Syarat 2: Pesan M yang sama dikirim ke 3 penerima (Alice, Bob, Carol) ✓

Syarat 3: gcd(Nᵢ, Nⱼ) = 1 untuk semua pasangan (CRT memerlukan ini)

gcd(N₁, N₂) = gcd(33, 85) = ?

33 = 3 × 11    85 = 5 × 17 → tidak ada faktor sama

gcd(33, 85) = 1 ✓ gcd(N₁, N₃) = gcd(33, 91) = ?

33 = 3 × 11    91 = 7 × 13 → tidak ada faktor sama

gcd(33, 91) = 1 ✓ gcd(N₂, N₃) = gcd(85, 91) = ?

85 = 5 × 17    91 = 7 × 13 → tidak ada faktor sama

gcd(85, 91) = 1 ✓
Semua syarat terpenuhi! Serangan bisa dilakukan.
Langkah 2 — Hitung N gabungan

Hitung N = N₁ × N₂ × N₃

Dalam CRT, kita butuh N (produk semua modulus). Ini adalah "ruang" dimana M³ berada secara eksak (bukan modular).

N = N₁ × N₂ × N₃ N = 33 × 85 × 91 N = 33 × 85 = 2,805 N = 2,805 × 91 = 255,255 N = 255,255
Kenapa N penting? Karena M³ < N₁N₂N₃ = 255,255. Artinya M³ tidak "terpotong" oleh mod, dan CRT akan mengembalikan nilai M³ yang sebenarnya!

Syarat: M < min(Nᵢ) = 33, jadi M³ < 33³ = 35,937 < 255,255 ✓

Langkah 3 — Hitung Mᵢ (faktor CRT)

Hitung Mᵢ = N / Nᵢ untuk tiap penerima

Mᵢ adalah N dibagi dengan modulus penerima ke-i. Ini bagian dari formula CRT.

M₁ = N / N₁ = 255,255 / 33 = 7,735 M₂ = N / N₂ = 255,255 / 85 = 3,003 M₃ = N / N₃ = 255,255 / 91 = 2,805
Mᵢ ini nanti akan di-mod dengan Nᵢ untuk mencari invers modularnya (yᵢ).

Formula CRT lengkap:

X = (C₁·M₁·y₁ + C₂·M₂·y₂ + C₃·M₃·y₃) mod N

X adalah M³ yang sebenarnya. Langkah berikutnya adalah mencari yᵢ.

Langkah 4 — Hitung yᵢ (invers modular)

Hitung yᵢ = Mᵢ⁻¹ mod Nᵢ menggunakan Extended Euclidean

Kita perlu mencari yᵢ sedemikian sehingga Mᵢ · yᵢ ≡ 1 (mod Nᵢ). Caranya: hitung Mᵢ mod Nᵢ dulu, baru cari inversnya.

y₁ = M₁⁻¹ mod N₁ = 7735⁻¹ mod 33

7735 mod 33 = 7735 - 234×33 = 7735 - 7,722 = 13

Jadi kita cari: 13⁻¹ mod 33 → Extended Euclidean:

33 = 2×13 + 7 13 = 1×7 + 6 7 = 1×6 + 1 ← gcd = 1 Balik: 1 = 7 - 6 = 7 - (13 - 7) = 2×7 - 13 = 2×(33-2×13) - 13 = 2×33 - 5×13 → 13⁻¹ ≡ -5 ≡ 28 (mod 33) y₁ = 28

y₂ = M₂⁻¹ mod N₂ = 3003⁻¹ mod 85

3003 mod 85 = 3003 - 35×85 = 3003 - 2,975 = 28 85 = 3×28 + 1 ← gcd = 1 → 1 = 85 - 3×28 → 28⁻¹ ≡ -3 ≡ 82 (mod 85) y₂ = 82

y₃ = M₃⁻¹ mod N₃ = 2805⁻¹ mod 91

2805 mod 91 = 2805 - 30×91 = 2805 - 2,730 = 75 91 = 1×75 + 16 75 = 4×16 + 11 16 = 1×11 + 5 11 = 2×5 + 1 ← gcd = 1 Balik: 1 = 11 - 2×5 = 11 - 2(16-11) = 3×11 - 2×16 = 3(75-4×16) - 2×16 = 3×75 - 14×16 = 3×75 - 14(91-75) = 17×75 - 14×91 → 75⁻¹ ≡ 17 (mod 91) y₃ = 17
Langkah 5 — Hitung X menggunakan CRT

Susun dan hitung X = (C₁M₁y₁ + C₂M₂y₂ + C₃M₃y₃) mod N

Sekarang kita punya semua komponen. Masukkan ke formula CRT:

iCᵢMᵢyᵢCᵢ × Mᵢ × yᵢ
1267,7352826 × 7,735 × 28 = 5,631,080
2403,0038240 × 3,003 × 82 = 9,849,840
3342,8051734 × 2,805 × 17 = 1,621,290
Jumlah = 5,631,080 + 9,849,840 + 1,621,290 = 17,102,210 X = 17,102,210 mod 255,255 17,102,210 ÷ 255,255 = 67 sisa 125 67 × 255,255 = 17,102,085 17,102,210 - 17,102,085 = 125 X = 125
X = 125 adalah nilai M³ yang sebenarnya (bukan modular)! Sekarang tinggal akar pangkat 3.
Langkah 6 — Ambil akar pangkat e

Hitung M = X^(1/e) = 125^(1/3)

Kita tahu X = M³ dan X = 125. Kita perlu mencari M dengan mengambil akar pangkat 3 secara integer (bukan float!).

X = M^e = M³ = 125 M = ³√125 = ? Cek: 1³ = 1, 2³ = 8, 3³ = 27, 4³ = 64, 5³ = 125 ✓ M = 5
Penting: ini adalah integer cube root, bukan floating point! Gunakan binary search atau Newton's method untuk angka besar.
Langkah 7 — Verifikasi

Cek: apakah M=5 benar?

Enkripsi ulang M=5 dengan kunci publik masing-masing penerima dan bandingkan dengan ciphertext asli:

C₁ = M³ mod N₁ = 5³ mod 33 = 125 mod 33 125 = 3×33 + 26 → C₁ = 26 ✓ C₂ = M³ mod N₂ = 5³ mod 85 = 125 mod 85 125 = 1×85 + 40 → C₂ = 40 ✓ C₃ = M³ mod N₃ = 5³ mod 91 = 125 mod 91 125 = 1×91 + 34 → C₃ = 34 ✓
Semua ciphertext cocok! Serangan berhasil.
Plaintext ditemukan
M = 5

Ringkasan langkah Hastad

1. Verifikasi syarat (e kecil, pesan sama, gcd=1)

2. Hitung N = N₁ × N₂ × N₃

3. Hitung Mᵢ = N / Nᵢ

4. Hitung yᵢ = Mᵢ⁻¹ mod Nᵢ (Extended Euclidean)

5. Hitung X = (ΣCᵢMᵢyᵢ) mod N

6. Hitung M = X^(1/e)

7. Verifikasi
