Soal — Multiprimes Attack

Pecahkan RSA dengan 3 faktor prima!

Seorang developer ceroboh membuat RSA dengan N = p × q × r (tiga faktor prima) dan faktor-faktornya relatif kecil. Eve mendapatkan kunci publik dan ciphertext berikut:

Langkah 1 — Pahami perbedaan Multiprimes vs RSA biasa

RSA biasa vs RSA multiprime

RSA biasa (2 faktor)
N = p × q φ(N) = (p-1)(q-1) d = e⁻¹ mod φ(N)
RSA multiprime (3 faktor)
N = p × q × r φ(N) = (p-1)(q-1)(r-1) d = e⁻¹ mod φ(N)
Perbedaan KRITIS: φ(N) melibatkan tiga faktor, bukan dua. Kalau salah pakai rumus φ(N) = (p-1)(q-1), d yang didapat salah dan dekripsi gagal!
Kenapa lemah? Untuk N k-bit dengan 3 faktor, tiap faktor hanya ≈ k/3 bit → jauh lebih kecil → lebih mudah ditemukan dengan trial division.

Strategi serangan: cukup temukan SATU faktor prima saja dari N, sisanya bisa dibagi langsung.

Kalau ketemu p → q×r = N/p → faktorkan q×r untuk dapat q dan r

Langkah 4 — Hitung φ(N) multiprime

φ(N) = (p-1)(q-1)(r-1) — BUKAN (p-1)(q-1)!

Ini perbedaan paling krusial dari RSA biasa. Karena N punya 3 faktor prima:

φ(N) = (p-1) × (q-1) × (r-1) φ(2431) = (11-1) × (13-1) × (17-1) = 10 × 12 × 16 = 120 × 16 φ(N) = 1,920
Jebakan umum di soal: kalau salah pakai φ(N) = (p-1)(q-1) = 10×12 = 120 → d yang didapat salah → dekripsi gagal!
Rumus umum: untuk N = p₁ × p₂ × ... × pₖ, maka φ(N) = (p₁-1)(p₂-1)...(pₖ-1)
Langkah 5 — Hitung kunci privat d

d = e⁻¹ mod φ(N) = 7⁻¹ mod 1920

Cari d sehingga 7d ≡ 1 (mod 1920). Gunakan Extended Euclidean:

Fase maju:

1920 = 274 × 7 + 2 7 = 3 × 2 + 1 2 = 2 × 1 + 0 ← gcd = 1

Fase mundur (substitusi balik):

1 = 7 - 3 × 2 = 7 - 3 × (1920 - 274 × 7) = 7 - 3×1920 + 822×7 = 823×7 - 3×1920 7⁻¹ ≡ 823 (mod 1920) → d = 823

Verifikasi:

7 × 823 = 5,761 5,761 mod 1920 = 5761 - 2×1920 = 5761 - 3840 ... 5761 - 3×1920 = 5761 - 5760 = 1 ✓
d = 823 adalah kunci privat yang berhasil direkonstruksi!

Ringkasan langkah Multiprimes Attack

1. Pahami: N = p×q×r, φ(N) = (p-1)(q-1)(r-1)

2. Trial division pada N → temukan faktor pertama p

3. Faktorkan N/p → temukan q dan r

4. Hitung φ(N) = (p-1)(q-1)(r-1)

5. Hitung d = e⁻¹ mod φ(N) (Extended Euclidean)

6. Dekripsi M = C^d mod N (fast modular exp / CRT)

7. Verifikasi: C = M^e mod N
